# Customer Service AI

A small, professional AI-powered customer service assistant built with Python, Streamlit, and the Anthropic Claude API. Designed as a clean, beginner-friendly portfolio project that demonstrates practical, real-world AI implementation skills.

## Description

Customer Service AI is a web app that simulates a support assistant for a fictional outdoor gear company, **Northstar Outfitters**. Users can ask common customer support questions (shipping, returns, warranty, order tracking, troubleshooting, etc.) and receive answers generated by Claude, grounded in a small local knowledge base.

## What the App Does

- Accepts a customer support question typed in by the user
- Lets the user optionally select a support category (Shipping, Returns, Warranty, Orders, Troubleshooting, General)
- Sends the question, along with relevant knowledge base content, to the Claude API
- Returns a clear, friendly, accurate answer based only on the knowledge base
- Detects when a question should be escalated to a human agent and says so
- Lets the user copy or download the generated response

## Why This Project Exists

This project was built to demonstrate practical skills in:

- Designing AI prompts that stay grounded in real source data (not hallucinated answers)
- Structuring a small AI application cleanly, the way a junior AI engineer would in a real job
- Building a simple but professional Streamlit interface
- Implementing basic escalation logic, a common requirement in real support AI systems
- Writing clear, professional documentation suitable for a public GitHub portfolio

It follows the same project structure and documentation style as a previous portfolio project, `resume-critiquer`.

## Features

- Simple, clean Streamlit chat-style interface
- Local knowledge base (Markdown or JSON) covering core support topics
- Claude-generated answers grounded strictly in the knowledge base
- Escalation detection for questions outside the knowledge base or requiring a human
- Example questions to help users try the app quickly
- Copy/download support response
- No database, no login, no real customer data вЂ” safe to demo publicly

## Setup Instructions

1. Clone the repository
   ```bash
   git clone https://github.com/wyzd0m/customer-service-ai.git
   cd customer-service-ai
   ```
2. Create a virtual environment
   ```bash
   python -m venv venv
   source venv/bin/activate   # On Windows: venv\Scripts\activate
   ```
3. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```
4. Set up your environment variables (see below)
5. Run the app
   ```bash
   streamlit run app.py
   ```

## Environment Variables

Create a `.env` file in the project root (never commit this file):

```
ANTHROPIC_API_KEY=your_api_key_here
```

A `.env.example` file is included in the repo to show the required format without exposing real keys.
