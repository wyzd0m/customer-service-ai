# Features

## MVP Features

- Customer question input (free text)
- Single fictional company: Northstar Outfitters
- Support category selection (Shipping, Returns, Warranty, Orders, Troubleshooting, General)
- Claude-generated answer to the customer's question
- Knowledge-base-grounded responses (no invented policies)
- Escalation suggestion when the question can't be confidently answered
- Copy or download the generated support response
- Example questions to help users try the app quickly

## Nice-to-Have Features

- Light/dark theme toggle
- Category-based example question suggestions
- Response tone selector (e.g., formal vs. casual)
- Simple loading/typing indicator while Claude generates a response
- Basic markdown formatting in responses (bullet points, bold text)

## Future Advanced Features

- Ticket creation simulation
- Customer sentiment detection
- Conversation history across a session
- FAQ upload (let a user add their own knowledge base files)
- RAG / vector search for larger knowledge bases
- Admin knowledge base editor (add/edit/delete entries via UI)
- Email draft generation for support agents
- CRM integration mock
- Multi-company support (switch between fictional businesses)
- Authentication and persistent storage
