# Return Policy

## Standard Returns
- Items can be returned within 30 days of the delivery date
- Items must be unused, with original tags attached, in original packaging
- Items marked "Final Sale" or purchased from the clearance section are not eligible for return

## How to Return an Item
1. Start a return from the "Order Status" page using the order number and email
2. Print the prepaid return shipping label (free for US customers)
3. Drop the package off at any carrier location listed on the label

International customers are responsible for return shipping costs.

## Damaged or Incorrect Items
If an item arrives damaged, defective, or is not what was ordered:
- Contact support within 14 days of delivery
- Include the order number and a photo of the issue if possible
- Northstar Outfitters will cover return shipping in these cases and send a
  replacement or full refund, customer's choice

## Refund Timeline
- Refunds are issued to the original payment method
- Once a return is received and inspected, refunds are processed within 5-7 business days
- It may take an additional 2-5 business days for the refund to appear on the customer's
  statement, depending on their bank or card issuer

## Exchanges
Direct size or color exchanges are available for most items. Customers can
request an exchange instead of a refund during the return process.
